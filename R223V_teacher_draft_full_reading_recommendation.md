# R223V teacher draft full reading recommendation

## Problem

R223T uses shortened teacher draft excerpts in the center panel. This is acceptable for sandbox review, but a review-only pilot should let reviewers read full teacher manuscripts without losing context.

## Recommendation

Future R223W spec should require:

- teacher draft as the primary document-style reading pane
- full manuscript view available before ledger inspection
- no field wall in the teacher draft
- no component shelf in the teacher draft
- no publish/save/apply controls near the draft

## Reading order

Recommended review order:

```text
safety banner
sample identity
teacher default draft full reading
v0.1 / v0.2 difference summary
review ledger summary
full ledger only by reviewer action
```

## Hold condition

Hold if a future pilot makes the teacher draft secondary to route metadata or ledger fields.
